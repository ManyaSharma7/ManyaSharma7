<summary><strong>Integrated Farm Intelligence System</strong> — Multi-Module Agricultural ML Platform</summary>

<br/>

A production-grade agricultural intelligence platform that combines five distinct machine learning modules to address crop recommendation, disease classification, water usage optimization, and farm clustering at scale. Developed in collaboration with Niharika Prasad for CSE274 (Applied Machine Learning) under the guidance of Dr. Dimple Nagpal.

| **Scale** | Multi-dataset ingestion with preprocessing, cross-validation, and model persistence |
| **Performance** | GridSearchCV-optimized hyperparameters; SHAP-based feature attribution for interpretability |
| **Security** | Input validation at API layer; schema-enforced request/response contracts |
| **Impact** | Demonstrates end-to-end ML product engineering from raw data to deployable REST service |
